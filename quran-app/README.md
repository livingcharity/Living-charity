# تطبيق القرآن الكريم

تطبيق ويب متقدم لقراءة القرآن الكريم مع التفسير وأسباب النزول.

## الملفات الأساسية

- **index.html** - الواجهة الرئيسية
- **app.js** - منطق التطبيق
- **style.css** - تصميم التطبيق

## مجلد البيانات

```
data/
├── quran/
│   └── quran-hafs-usmani.json (نص القرآن بالرسم العثماني)
├── tafsir/
│   └── tafsir.json (التفسير)
└── asbab-al-nuzul/
    └── asbab.json (أسباب النزول)
```

## الاستخدام

1. افتح `index.html` في المتصفح
2. اختر سورة من القائمة الجانبية
3. اضغط على أي آية لعرض التفسير وسبب النزول

## المميزات

✅ واجهة استجابة على جميع الأجهزة
✅ نصوص عربية صحيحة بدون تعديل
✅ ربط التفسير مع الآيات
✅ دعم RTL للنصوص العربية
✅ بدون اعتماديات خارجية
