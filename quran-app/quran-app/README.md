# Quran Application (Uthmani Script)

## الهدف
نشر القرآن الكريم بالرسم العثماني المعتمد، مع تفسير موثوق وأسباب نزول، بدون أي تعديل أو اجتهاد.

## المصادر
- النص القرآني: مجمع الملك فهد
- التفسير: ابن كثير – السعدي
- أسباب النزول: الواحدي – السيوطي
- الترجمات (لاحقًا): مجمع الملك فهد

## ملاحظات شرعية
- النص العربي قرآن كريم لا يُعدّل
- الترجمات هي "ترجمة معاني" فقط
- يُذكر سبب النزول فقط إن ثبت، وإلا يُصرّح بعدم وجوده

- يعمل Offline
- بدون ذكاء اصطناعي
